osom ui

## Setup Yarn v2 (Berry)

```shell
./yarn set version berry
```

## Compiles and minifies for production

```shell
./yarn build
```

## Style check

```shell
./yarn style
```

## Lints and fixes files

```shell
./yarn lint
```

## Run your unit tests

```shell
yarn test
```
